## 产品功能

**每日精选机会**
每天聚合来自 Hacker News、GitHub、Product Hunt、36氪、IndieHackers 的最新动态，经结构化研判后整理为当日副业机会，每条均有真实市场信号支撑、可追溯来源。

**机会详情**
每个机会包含核心痛点、市场分析、4 步执行路径、收益模式、启动成本、回收周期、竞争程度等维度，面向 1 人独立执行设计，3 个月内可见收益。

**今日技术风口**
展示驱动当天机会的 GitHub 趋势项目，了解背后的技术信号。

**帮助度投票**
登录后可对每个机会投票，反馈哪些机会真正有价值。

**邮件日报订阅**
订阅后每天将精选机会推送至邮箱，不错过任何一天。

**历史记录浏览**
可随时回溯任意日期的机会列表。
