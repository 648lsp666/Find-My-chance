# 每日机会雷达 · 产品定位

## 一句话定位

> **每天追踪真实市场信号，筛出有路径、成本、竞争度的副业机会——不是灵感，是执行起点。**

---

## 目标用户

### 核心用户（Primary）
**在职打工人，想开辟第二收入来源**
- 年龄：22–35 岁
- 收入：月薪 10k–40k
- 特征：有一定技能（代码/设计/写作），但没时间每天研究市场
- 核心痛点：知道要做副业，但不知道做什么、怎么开始
- 决策障碍：不知道投入产出比、启动成本是否可接受

### 次要用户（Secondary）
- **独立开发者**：寻找下一个 SaaS idea，需要市场信号
- **内容创作者**：寻找有价值的垂直选题，早于大众发现趋势

---

## 竞品分析与差异化

| 产品 | 内容形式 | 可操作性 | 更新频率 | 结构化 | 缺点 |
|------|---------|---------|---------|------|------|
| 小红书副业号 | 图文/视频 | 低（无路径） | 不定期 | 无 | 信噪比极低，夹带广告 |
| 知识星球副业圈 | 帖子 | 中 | 用户驱动 | 无 | 付费墙高，质量参差不齐 |
| 36kr / 虎嗅 | 深度文章 | 极低 | 日更 | 无 | 企业/投资视角，个人无法执行 |
| Product Hunt | 产品发现 | 低（无执行指导） | 日更 | 无 | 英文，非副业收入导向 |
| **每日机会雷达** | **结构化卡片** | **高（含执行路径）** | **每日更新** | **JSON 可筛选** | 需建立信任 |

**护城河来源：**
1. **数据结构化** — 每个机会有统一 Schema，可查询、可比较、可归档
2. **信号研判深度** — 非简单搬运，综合多个信号源（GitHub Trending、Product Hunt、国内社区）
3. **执行导向** — 每个机会包含：路径 + 成本 + 时间 + 竞争度，不是"灵感"是"起点"
4. **每日更新 + 归档** — 形成时间序列，可以看趋势演变

## 价值主张

```
对于 想做副业但不知道做什么的打工人
每日机会雷达 是一个基于多源信号的副业机会发现工具
它每天12点整理 GitHub Trending、Product Hunt 和国内热点
为你提供 3–8 个结构化机会卡片，每个包含执行路径、启动成本、竞争程度
不同于 小红书/知识星球的散乱内容
我们的差异在于 可筛选、有路径、每日更新、完全免费
```

## 品牌方向

- **名称**：每日机会雷达 / Opportunity Radar
- **色调**：深色暗调（当前设计） + 琥珀色 Accent — 传递「专业、前沿、工具感」
- **调性**：不贩卖焦虑，不夸大收益；客观、克制、有信息密度
- **反面**：避免「月入过万」「轻松赚钱」等焦虑营销话语

## 阶段目标

| 阶段 | 时间 | 核心指标 |
|------|------|---------|
| 内测 | 当前 | 个人使用，验证内容质量 |
| 公测 | 第1–2周 | 500 UV/日，留存 > 20% |
| 增长 | 第3–4周 | 1000 UV/日，邮件订阅 > 200 |
